STARTBUY E-COMMERCE WEB SITE

My project, STARTBUY, is an online commerce platform designed to provide a convenient and accessible shopping experience through the web. By leveraging modern technologies such as Angular, Spring Boot, and MySQL, STARTBUY offers users the ability to browse a wide range of products, place orders, and enjoy a seamless shopping experience. Angular's user-friendly interface ensures intuitive navigation, while the robustness of Spring Boot. The MySQL database management system ensures efficient management of stocks and orders.

APASFAX WEBSITE

Developed a website for APA Sfax during my schooling as an introductory project in web development. Utilizing HTML, CSS, and JavaScript, I created a platform highlighting the organization's efforts in safeguarding animal welfare in Sfax. The website was crafted with a user-friendly interface and responsive design, providing information about the organization's initiatives, adoption programs, and ways to support their cause. Implemented basic interactive elements using JavaScript to enhance the site's functionality and navigation. This project served as an initiation into web development, offering practical experience in coding and design while supporting a noble cause.
